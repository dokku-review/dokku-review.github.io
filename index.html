01How it works

  1. You install once

    One curl line on a fresh Ubuntu box. Dokku, Postgres, Let's Encrypt, Redis — all configured. Five minutes.

  2. CI deploys on each PR

    Label a PR review. The reusable GitHub Actions workflow builds your app and pushes it to pr-N.your-domain. A bot comment links the URL.

  3. Postgres is automatic

    The system-postgres plugin auto-provisions a Postgres database per PR app from a shared instance. Fast, cheap, no per-PR container overhead.

  4. Close the PR, it disappears

    The teardown workflow destroys the app and its database when the PR closes. GitHub deployments get marked inactive. Clean slate.

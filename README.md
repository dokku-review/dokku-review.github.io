# dokku-review.github.io

Source for <https://dokku-review.github.io> — landing site, install wizard, and CI walkthrough for [dokku-review](https://github.com/dokku-review), Heroku-style review apps on your own Dokku server.

Plain static HTML, no build step. Preview locally with any static server, e.g. `python3 -m http.server` then open `localhost:8000`.

## Related repos

- [`dokku-review/actions`](https://github.com/dokku-review/actions) — reusable GitHub Actions workflows (deploy / teardown)
- [`dokku-review/system-postgres`](https://github.com/dokku-review/system-postgres) — Postgres review-DB plugin
